
<photowall.html>
    1. 双击图片区域选择文件,可选多个文件
    2. 可鼠标拖拽、滚轮缩放图片
    3. 图片拖拽可互换位置
    4. 如果有重复图片，按下鼠标时会有闪烁提示
    5. chrome谷歌浏览器截屏方法
        1. F12
        2. crtl + shift + P
        3. 输入 Capture full size screenshot
        4. Enter回车开始截屏，等待几秒

       Microsoft Edge浏览器截屏方法
        1. 点击浏览器右上角“添加笔记”
        2. 点击新窗口右上角“剪辑”(截图)
        3. 框选截图
